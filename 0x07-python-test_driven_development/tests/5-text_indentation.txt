'text indentation module'
==================================
indents text with two newlines after every encounter with '.' '?' ':' characters

>>> text_indentation = __import__('5-text_indentation').text_indentation

Normal Case
>>> text_indentation("This is the first normal sentence. This one does not end in punctuation")
This is the first normal sentence.
<BLANKLINE>
This one does not end in punctuation

Case text ends in punction
>>> text_indentation("This is the first normal sentence. This one does end in punctuation.")
This is the first normal sentence.
<BLANKLINE>
This one does end in punctuation.
<BLANKLINE>

Case text is not a valid string
>>> text_indentation([1, 2, 3])
Traceback (most recent call last):
TypeError: text must be a string

Case text empty
>>> text_indentation("")

Case text param missing
>>> text_indentation()

Case consecutive chaaracters in text
>>> text_indentation("Hello.?Is it me..You are looking for:?.")
Hello.
<BLANKLINE>
?
<BLANKLINE>
Is it me.
<BLANKLINE>
.
<BLANKLINE>
You are looking for:
<BLANKLINE>
?
<BLANKLINE>
.
<BLANKLINE>

Case text has special character
>>> text_indentation("Do you know my name?This is how it is written: Sanelisiwe.\nWow, that's pretty")
Do you know my name?
<BLANKLINE>
This is how it is written:
<BLANKLINE>
Sanelisiwe.
<BLANKLINE>
Wow, that's pretty

Case text has no occurence of characters
>>> text_indentation("This sentence does not have the characters")
This sentence does not have the characters

Case text has spaces next to characters
>>> text_indentation("This sentence has a space before the character . And this one after?")
This sentence has a space before the character .
<BLANKLINE>
And this one after?
<BLANKLINE>

 Case text has characters at the beginning and end of text
 >>> text_indentation(":This sentence has characters at the beginning and end.")
 :
 <BLANKLINE>
 This sentence has characters at the beginning and end.
 <BLANKLINE>

Case text beings with a space
>>> text_indentation(" This sentence begins with a space. It needs to be removed")
This sentence begins with a space.
<BLANKLINE>
It needs to be removed
